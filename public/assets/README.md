# Asset Directory Guide

Masukkan asset sesuai nama file berikut agar mudah dipakai di komponen Next.js.

## Brand
- `brand/logo.png`
- `brand/mitsuru.png`
- `brand/favicon.png`

## Games
Setiap game memakai struktur:

```text
games/{slug}/icon.png
games/{slug}/poster.jpg
games/{slug}/banner.jpg
```

Slug yang disiapkan:
- `mobile-legends`
- `free-fire`
- `pubg-mobile`
- `valorant`
- `genshin-impact`
- `honor-of-kings`
- `roblox`
- `steam-wallet`
- `tiktok-live`
- `bigo-live`
- `joki-rank`
- `magic-chess`
- `delta-force`
